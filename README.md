# ClaimsAI: AI-Powered Car Insurance Claims Assessment

A functional prototype demonstrating an AI-assisted car insurance claims workflow. Built for a claims agent persona, the app automates damage assessment from uploaded vehicle photos, generates structured cost estimates, and provides a human-in-the-loop review and approval interface.

> **Note:** This is a prototype with simulated AI responses.

---

## Quick Start

```bash
# Clone the repo
git clone https://github.com/<your-username>/claimsai.git
cd claimsai

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
